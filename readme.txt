Plugin that uses Brown Paper Tickets API for Fair Trade Ticketing


== Description ==

This plugin pulls your events from brownpapertickets.com and will list all of them and the event date and time and provide a link for your users to buy tickets off of BPT

== Installation ==

1. Upload the `wp-bpt` folder to the `/wp-content/plugins/` directory
1. Activate the Brown Paper Tickets plugin through the 'Plugins' menu in WordPress
1. You MUST add your BPT credentials to the Plugin Settings page for the plugin to work
